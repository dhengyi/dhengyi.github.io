Java网络爬虫--HttpClient设置头部信息与模拟登录策略

在网络爬虫中我们经常需要设置一些头部信息，使我们进行网页抓取的行为更加像使用浏览器浏览网页一般，并且我们有时需要将头部信息设置正确，才能得到正确的数据，要不然有可能得到的信息和浏览器所展示的页面有出入。

设置头部还可以进行模拟登录，我们可以设置Cookie，来得到登录后的页面，来抓取我们需要的数据。
